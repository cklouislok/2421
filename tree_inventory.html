Tree Inventory Map of the City of Peterborough

This product is an interactive web map of trees in the City of Peterborough designed for public use.

The product is configured with major reference to ArcGIS Maps SDK for JavaScript Version 4.29, and also some reference to external sources such as sample codes for Cascading Style Sheets (CSS).

Main functions

Check out our product, our ArcGIS StoryMaps or the descriptions below for a tour of its features!

Feature 1: Popup Information for Individual Trees

Zoom in and click on any tree to access information like status, common name, and location!

Feature 2: Tree Status Selector

Users can select a particular status of trees to display in the map. This function is constructed by Structured Query Language (SQL).

Feature 3: Tree Zone Statistics

Zoom out and click on any tree zone to look at five most common species of that zone. The numbers change dynamically based on the tree status selection.

Source

Tree Inventory and Tree Zone data obtained from the City of Peterborough.

Other Design Elements

Responsive webpage design, customized loading screen, how-to-use modal button and useful widgets.

Limitation

Due to data security and privacy concerns, the potential planting area data layer created in this project cannot be incorporated into this product.

Recommendation

If time allows, the popup information for indiviudal trees can include links to more information, photos, or other useful information for the public.
